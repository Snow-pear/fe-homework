## 使用步骤

1. Fork此项目。
2. 本地开发运行：

```shell
cd $工作目录
git clone 刚才fork的仓库地址
cd fe-homework
npm install
npm run start
```

3. 创建目录，完成作业。
4. commit修改，并push到github。
